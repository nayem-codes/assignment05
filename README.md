<h3><b>1. Difference between var, let, and const:</b></h3> 
<b>var:</b> var is an old way of declaring variables. It can be reassigned as well as redeclared.
<br>
<b>let:</b> let was introduced in ES6. It can be reassigned but can not be redeclared.
<br>
<b>const:</b> cosnt was also introduced in ES6. It can neither be reassigned nor redeclared.
<br>
<h3><b>2. Spread Operator:</b></h3> 
We use spread operator when accessing the values inside an array, also to assign the values of an array to parameters serially. Besides let's say if there's two non primitive data types where one is made of other one and we want to change value in one data only, in this case we can use spread operator to prevent these two data take reference of changing value. So we can use this operator to make an array with an array.
<h3><b>3. Difference between map(), filter(), and forEach():</b></h3>
<b>map():</b> It is used for creating an array by applying a function to each element of the original array.
<br>
<b>filter():</b> It is used for creating an array with only those elements that fullfil the given condition.
<br>
<b>forEach():</b> It is used for applying a function on every element of an array.
<br>
<h3><b>4. Arrow Function:</b></h3>
Arrow function is a shorter way of writing fuctions. It has a few difference with normal function. For example, Arrow function can be written in a single line. Also if there is one parameter, using parentheses is optional.
<br>
<h3><b>5. Template Literals:</b></h3>
Template Literals are made of backticks(``). It is used for creating strings more dynamically by using ${}. It allows to write multiple lines.
